[03] External: DCode

Primary Tool: DCode.fr

A comprehensive toolkit for obscure ciphers and symbol-based encryption (Zodiac, Pigpen, etc).

WHEN TO USE: Encounter symbols or non-alphanumeric characters that don't fit standard English ciphers.

[04] External: MultiDecoder

Primary Tool: CacheSleuth MultiDecoder

An all-in-one decoding station for Base64, Hex, Binary, and ROT variations.

WHEN TO USE: Use this when you have a string of data but you are unsure what encoding it uses. It will attempt multiple decodings at once.

[05] External: CrackStation

Primary Tool: CrackStation Hash Cracker

Uses massive lookup tables (rainbow tables) to reverse-engineer password hashes.

WHEN TO USE: Use this when you have recovered a SHA1, MD5, or SHA256 hash and need to find the original plaintext password.

[06] External: Online EXIF Viewer

Primary Tool: EXIF Inspector

Extracts metadata from image files, including GPS coordinates and camera serial numbers.

WHEN TO USE: Use this during "Geoint" (Geospatial Intelligence) investigations to find where and when a photo was taken.

[07] External: EZGIF Metadata

Primary Tool: EZGIF View-Metadata

A lightweight tool to strip or view metadata from various file formats including GIFs and PNGs.

WHEN TO USE: Use this for quick inspection of web-based images to see if data was hidden or if the file was recently edited.

[08] External: Metadata2Go

Primary Tool: Metadata2Go All-File Inspector

Supports a huge range of file types, including DOCX, PDF, and Video files, to reveal hidden ownership data.

WHEN TO USE: Use this for non-image files. It is the best tool for finding the "Author" or "Creation Date" of a document or video.
